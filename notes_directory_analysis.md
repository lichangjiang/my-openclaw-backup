3. **notes 目录混合了多种类型的内容**

   - WSL 环境配置
   - 工作草稿

## 🧹 清理建议

### 高优先级清理

**1. 删除 mypy 缓存（87 MB）**
```bash
rm -rf ~/onedrive/notes/.mypy_cache
```
**原因：**
- 缓存文件可以重新生成
- 不应该同步到 OneDrive
- 占用 87 MB 空间

**2. 清理模型保存文件**
- `python/jp/rloh/ch12/saves/` 目录
- 如果不需要保留训练过程，可以删除旧的 checkpoint

### 中优先级清理

**3. 移动 WSL 环境配置到其他位置**
- JDK、Maven 等工具不应该在 notes 目录
- 建议移动到 `env/` 或单独目录

**4. 移动数据集到单独位置**
- FashionMNIST 等数据集不应该在 notes 目录
- 建议移动到 `data/` 或 `datasets/` 目录

### 低优先级

**5. 清理草稿目录**
- 检查草稿文件是否还需要
- 删除过时的草稿
