## 💡 查看下载速度的方法

### 方法 1：使用 journalctl 查看实时日志

```bash
# 实时查看同步日志
journalctl --user -u onedrive -f --no-pager
```

**统计最近 1 分钟的下载文件数：**
```bash
# 查看最近 1 分钟的下载速度
journalctl --user -u onedrive --since "1 minute ago" --no-pager | grep -c "Downloading file"
```

### 方法 2：使用自定义脚本监控速度

**创建监控脚本：**
```bash
cat > ~/monitor_onedrive_speed.sh << 'EOF'
#!/bin/bash
while true; do
    count=$(journalctl --user -u onedrive --since "1 minute ago" --no-pager | grep -c "Downloading file")
    echo "$(date '+%H:%M:%S') - 下载速度: $count 文件/分钟"
    sleep 60
done
EOF

chmod +x ~/monitor_onedrive_speed.sh
```

**运行监控脚本：**
```bash
~/monitor_onedrive_speed.sh
```

### 方法 3：使用 du 命令监控目录大小

```bash
# 每 5 分钟查看 OneDrive 目录大小
watch -n 300 'du -sh ~/onedrive/'
```

### 方法 4：使用 find 命令统计文件数

```bash
# 每小时统计一次新增文件数
while true; do
    current=$(find ~/onedrive -type f | wc -l)
    echo "$(date '+%H:%M:%S') - 总文件数: $current"
    sleep 3600
done
```
